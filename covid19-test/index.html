法国新冠病毒核酸检测和血清特异性IgM抗体检测是什么，在哪里做？

作者：攻略君 2020-11-04
  • 疫情

根据中国驻法国使馆的最新公告，所有搭乘航班赴华人员须凭新冠病毒核酸检测和血清特异性IgM抗体检测阴性证明才能登机，两者缺一不可。那么，在法国的小伙伴如何检测，以及在哪里才能做这两个检测呢？

这两种检测是做什么用的？

新冠病毒核酸检测Les tests virologiques (RT-PCR)是用来检查被测试者是否患有新冠病毒的。这个检测主要针对的是具有新冠症状，或者和新冠患者有过密切接触的人的。

血清特异性IgM抗体检测（Tests de dépistage Covid-19）是检查被测试者是否曾经感染过病毒并已经产生抗体的。做这个检测的前提是你没有任何新冠症状。

核酸/血清检测注意事项

检测时间

官方公告上说的很清楚，必须在两天之内做检测。也就是说，如果你在1号做的检测，那么登机时间不能晚于3号。

检测机构

只要是法国卫生部门认可的，具备资质的检测机构都可以进行检测。小伙伴们可以在官网上查询离自己住所最近的监测点。这里需要注意的是，有的小伙伴说，药房也可以做检测，但是，并不给出具证明。因此打算回国的小伙伴还是要去周边的实验室检测的好，检测完毕后他们会出具证明的。

检测步骤

核酸检测

如果你具有新冠症状，或者和新冠患者有过密切接触的话，就需要进行核酸检测Les tests virologiques (RT-PCR)。

首先你需要医生的处方才能进行检测。可以参照前面提到的官方页面)查询具备资质的检测机构。强烈建议提前预约，否则会把时间都浪费在排队上。这个检测会在24小时出结果。这个检测在多数检测机构是免费的。个别的机构需要先付费再走医疗保险报销。

血清检测

你只要去就近药店，和工作人员说要做血清检测（Tests de dépistage Covid-19）。他们会询问你是否有新冠症状。如果你没有的话，就可以进行检测了。整个检测过程非常快，大约20分钟左右出结果。这项检测是自费的。费用大约为30欧元左右。
